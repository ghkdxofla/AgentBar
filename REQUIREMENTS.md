# LLM Agent의 사용량을 표시해주는 맥용 앱

Claude Code, Open AI Codex, Z.ai Coding Plan 3가지를 구독하고 있고 각 서비스의 실시간 사용량을 맥의 상단 메뉴바에 표시하고 싶어. 표시 공간을 적게 차지 하기 위해 5시간, 주간 사용량을 하나의 막대에서 색깔만 다르게 해서 표시하고 언급한 최대 3개의 서비스를 버티컬하게 표시하면 좋을 것 같아.
마우스오버 하면 상세한 정보가 나오고 그냥 두면 막대 그래프만 존재하도록 해. 최대 3개의 서비스를 쓰고 있지만 데이터가 없거나 구현이 불가능하다면 표시 가능한 그래프만 보여도 좋다. 맥북이 켜지면 자동으로 실행될 수 있게 해주고 설정에서 이 옵션을 끄거나 켤 수 있게 해.
